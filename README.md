## 動作法
```

端末2$ chmod +x count.py
端末2$ rosrun mypkg count.py
端末3$ rosnode list
端末3$ rostopic list
端末3$ rostopic echo /count_up
端末4$ chmod +x twice.py
端末4$ rosrun mypkg twice.py
```
